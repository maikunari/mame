Claude Code

New features & shipped skills

  1. Browser Harness — a self-healing browser framework

    Browser Harness is an open-source library that lets LLMs drive Chrome directly over CDP WebSocket with no intermediate framework. It includes a self-healing system that edits helper Python functions on-the-fly when tasks fail.

    Why it mattersDrop-in alternative to Playwright/Puppeteer wrappers when you need a browser that adapts to the page instead of the other way around — useful for agent-driven scraping and QA pipelines Mike already runs.

    via X April 18, 2026

  2. /autofix-pr — Cursor's CI-loop command from the terminal

    /autofix-pr is a Cursor command that lets you trigger the cloud-based Autofix PR workflow directly from your terminal after pushing a branch. It uploads your full session context so the autofixer can resolve CI failures, address review comments, and iterate without manual intervention.

    Why it mattersCloses the loop on a PR in one CLI step instead of switching to the web UI, tightening the local-to-cloud dev workflow between "done coding" and "shipped with green CI."

    via X April 17, 2026

  3. A developer built a local dashboard that parses the log files Claude Code already writes to disk, surfacing per-model and per-project token usage that Anthropic's official UI deliberately hides (only showing a single progress bar).

    Why it mattersPrecise observability into token spend across projects and models without relying on a black-box vendor dashboard — directly useful when wiring Claude into your own dev tools, agents, or billing-aware workflows.

    via X April 16, 2026

02

Design

Craft, typography, and what's on the page

  1. Drams — a Rams-inspired Framer component library

    A free collection of beautifully crafted, interactive Framer components (buttons, switches, sliders, dials, rotary radios, Braun-inspired shavers, OP-1 style keys) built with strict adherence to Dieter Rams' ten principles of good design. The site doubles as a remix platform where creators can submit and get featured.

    Why it mattersProduction-ready, skeuomorphic UI assets that can drop straight into Framer prototypes or serve as high-fidelity reference when auditing the tactile feel of internal tools.

    via X April 15, 2026

  2. Clone any site's rendered DOM with Claude + Chrome MCP

    A Claude-powered skill that uses Chrome's MCP to inspect a live website's rendered DOM, fonts, colors, and layout, then generates a pixel-perfect clone in a single prompt. The full implementation is open source on GitHub.

    Why it mattersPractical pattern for letting an LLM read the browser's live computed styles instead of guessing from screenshots — the kind of high-fidelity, tool-augmented workflow that beats "describe this design" by a wide margin.

    via X April 14, 2026
